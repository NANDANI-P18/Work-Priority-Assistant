# 📋 Work Priority Assistant

A simple and clean web application that helps you manage your tasks by priority and deadlines — so you always know what to work on first!

---

## 🌟 Features

- ✅ **Add Tasks** — Quickly create new tasks with just a few clicks
- 🔴 **Set Priority Levels** — Mark tasks as High, Medium, or Low priority
- 📅 **Deadline / Due Date** — Set due dates so you never miss a deadline
- ☑️ **Mark as Done** — Check off tasks when you complete them

---

## 🛠️ Technologies Used

- **HTML** — Structure of the website
- **CSS** — Styling and design of the website

## 🚀 How to Run This Project

Follow these simple steps to open the project on your computer:

1. **Download or Clone the Repository**
   ```
   git clone https://github.com/your-username/work-priority-assistant.git
   ```

2. **Open the Project Folder**
   - Go to the folder where you downloaded the project

3. **Open the Website**
   - Find the file called `index.html`
   - Double-click it — it will open directly in your browser!

> No installation or setup needed. It runs straight in your browser! 🎉
